## Description

Morning standup. Produces or updates the day's file with tasks from sprint plan, configurable chores from occasions.md (including day-specific chores), and blockers section. Takes <5 min. Accepts date or file path as input.

## Acceptance Criteria

- [x] Running /daily produces or updates day file (2026-wkNN-day.md)
- [x] Chores section populated from occasions.md including day-specific items
- [x] Tasks pulled from active sprint plan
- [x] File has valid Obsidian frontmatter with prev/next/up wikilinks
- [x] Idempotent — running twice on same day updates rather than overwrites
