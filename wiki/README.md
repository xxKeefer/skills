# Wiki

Raw note intake, knowledge management, and vault indexing for Obsidian.

## Skills

| Skill | Purpose |
|---|---|
| `/intake-raw` | Batch-process `00-raw/` notes -- classify, route, absorb-and-split |
| `/intake-it` | Single HITL note intake with human guidance |
| `/standardise-frontmatter` | Normalise knowledge note frontmatter to canonical shape |
| `/edit-article` | Structural and prose editing for Obsidian notes |
| `/rebuild-index` | Generate or rebuild INDEX.md for a directory |

## Installation

```json
{
  "enabledPlugins": {
    "xxkeefer-skills@wiki": true
  }
}
```

Requires `xxkeefer-skills@primitives`.
