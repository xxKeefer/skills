# Scribe

Capture and edit Obsidian vault notes -- procedure capture and prose editing.

## Skills

| Skill | Purpose |
|---|---|
| `/add-procedure` | Capture a just-solved task into the best-fit procedure file (or a new one) |
| `/edit-article` | Structural and prose editing for Obsidian notes |

## Installation

```json
{
  "enabledPlugins": {
    "scribe@xxkeefer-skills": true
  }
}
```

Requires `primitives@xxkeefer-skills`.
